about shortsites

the ultimate goal

the ultimate goal of shortsites is to provide a simple, easy to use platform for anyone to create and deploy websites in minutes without the complexity of troublesome hosting services.

how it works

shortsites gives you a code editor powered by monaco editor (by Microsoft) to write HTML, CSS, and JavaScript code in three monaco editor instances. Once you're happy with your site, you can deploy it in a single click with a unique URL that you can share easily. our backend is built on python, leveraging supabase to store website code and access keys.

for schools

shortsites is perfect for schools, as students can focus on learning web development concepts without worrying about complex deployment processes. Due to our one-click deployment process, teachers can easily review student work through the shared URLs, and the students can update their sites using our 'edit' feature.
