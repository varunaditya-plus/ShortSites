Introduction to JavaScript

JavaScript is a programming language (like python) that lets you create interactive web pages. It's an essential part of web applications and can update content, control the page, animate images, and much more.

JavaScript Syntax

function sayHello() {
    alert("Hello, World!");
}

sayHello();

JavaScript Basics

Variables

Variables are containers for storing data values. In JavaScript, you can declare variables using let or const.

In JavaScript, you can declare variables using let or const. The only difference is that while let can be changed (like in python), const cannot.

// let (can be changed)
let name = "John";
name = "Jane"; // This is allowed

// const (cannot be changed)
const age = 25;
age = 26; // This would cause an error

Data Types

JavaScript has several data types:

  • String - text values,  eg. "Hello"
  • Number - numeric values,  eg. 42 or 3.14
  • Boolean - true or false
  • Array - an ordered list of values

Functions

Functions are blocks of code designed to perform a particular task.

Functions can be declared using the function keyword, or as function expressions. A function declaration is hoisted, meaning it can be called before it is defined. A function expression is not hoisted, meaning it can only be called after it is defined.

// function declaration
function greet(name) {
    return "Hello, " + name + "!";
}

// function expression
const sayGoodbye = function(name) {
    return "Goodbye, " + name + "!";
};

// calling functions
console.log(greet("Alice")); // "Hello, Alice!"
console.log(sayGoodbye("Bob")); // "Goodbye, Bob!"

DOM Manipulation

The Document Object Model (DOM) is a way to access HTML page elements and use them in JavaScript code to change (or add/remove) the HTML elements, attributes, and CSS styles in the page.

// Selecting elements
const element = document.getElementById("myId");
const elements = document.getElementsByClassName("myClass");
const elementsByTag = document.getElementsByTagName("p");
const queryElement = document.querySelector(".myClass");
const queryElements = document.querySelectorAll("p");

// Changing HTML content
element.innerHTML = "New content";
element.textContent = "New text";

// Changing CSS
element.style.color = "blue";
element.style.fontSize = "20px";

// Adding/removing classes
element.classList.add("highlight");
element.classList.remove("old-class");
element.classList.toggle("active");

// Creating new elements
const newElement = document.createElement("div");
newElement.textContent = "I'm a new element";
document.body.appendChild(newElement);

Events

JavaScript can execute code when something happens. This 'something' is known as an event. An example of an even is when a user clicks a button.

// Adding event listeners
document.getElementById("myButton").addEventListener("click", function() {
    alert("Button was clicked!");
});

// Common events:
// click - when an element is clicked
// mouseover - when the mouse moves over an element
// mouseout - when the mouse moves away from an element
// keydown - when a key is pressed
// load - when the element has finished loading on the page
