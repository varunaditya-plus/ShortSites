Introduction to JavaScript

JavaScript is a programming language (like python) that lets you create interactive web pages. It's an essential part of web applications and can update content, control multimedia, animate images, and much more.

JavaScript Syntax

function sayHello() {
    alert("Hello, World!");
}

sayHello();

JavaScript Basics

Variables

Variables are containers for storing data values. In JavaScript, you can declare variables using var, let, or const.

// Using let (block-scoped, can be reassigned)
let name = "John";
name = "Jane"; // This is allowed

// Using const (block-scoped, cannot be reassigned)
const age = 25;
// age = 26; // This would cause an error

// Using var (function-scoped, older way)
var isStudent = true;

Data Types

JavaScript has several data types:

  • String - text values, e.g., "Hello"
  • Number - numeric values, e.g., 42 or 3.14
  • Boolean - true or false
  • Object - collections of related data
  • Array - ordered lists of values
  • Null - intentional absence of any value
  • Undefined - variable declared but not assigned a value

Functions

Functions are blocks of code designed to perform a particular task.

// Function declaration
function greet(name) {
    return "Hello, " + name + "!";
}

// Function expression
const sayGoodbye = function(name) {
    return "Goodbye, " + name + "!";
};

// Arrow function (ES6)
const add = (a, b) => a + b;

// Calling functions
console.log(greet("Alice")); // "Hello, Alice!"
console.log(sayGoodbye("Bob")); // "Goodbye, Bob!"
console.log(add(5, 3)); // 8

DOM Manipulation

The Document Object Model (DOM) is a programming interface for HTML documents. JavaScript can change all the HTML elements, attributes, and CSS styles in the page.

// Selecting elements
const element = document.getElementById("myId");
const elements = document.getElementsByClassName("myClass");
const elementsByTag = document.getElementsByTagName("p");
const queryElement = document.querySelector(".myClass");
const queryElements = document.querySelectorAll("p");

// Changing HTML content
element.innerHTML = "New content";
element.textContent = "New text";

// Changing CSS
element.style.color = "blue";
element.style.fontSize = "20px";

// Adding/removing classes
element.classList.add("highlight");
element.classList.remove("old-class");
element.classList.toggle("active");

// Creating new elements
const newElement = document.createElement("div");
newElement.textContent = "I'm a new element";
document.body.appendChild(newElement);

Events

JavaScript can execute code when events occur, such as when a user clicks a button.

// Adding event listeners
document.getElementById("myButton").addEventListener("click", function() {
    alert("Button was clicked!");
});

// Common events:
// click - when an element is clicked
// mouseover - when the mouse moves over an element
// mouseout - when the mouse moves away from an element
// keydown - when a key is pressed
// load - when the page has finished loading
// submit - when a form is submitted
