Introduction to CSS

CSS (Cascading Style Sheets) is used to style web pages to add color, and organisation. It controls how HTML elements are displayed on screen.

CSS Syntax

selector {
    property: value;
    property: value;
}

The selector points to the HTML element you want to style. The declaration block contains one or more declarations separated by semicolons.

CSS Selectors

CSS selectors are used to "find" the HTML elements on the page that you want to style.

Common Selectors:

  • Element Selector - selects elements based on the element name (e.g., p, h1, div)
  • ID Selector - uses the id attribute of an HTML element (#id)
  • Class Selector - selects elements with a specific class attribute (.classname)
  • Universal Selector - selects all HTML elements (*)
/* Element selector */
p {
    color: blue;
}

/* ID selector */
#header {
    background-color: black;
    color: white;
}

/* Class selector */
.highlight {
    background-color: yellow;
}

/* Universal selector */
* {
    margin: 0;
    padding: 0;
}

Common CSS Properties

As you will see, most styles are pretty self-explanatory, so this shouldn't be too hard.
Here are some of the most commonly used CSS properties:

  • color - sets the color of text
  • background-color - sets the background color of an element
  • font-family - specifies the font for text
  • font-size - sets the size of text
  • margin - sets the margin area on all four sides of an element
  • padding - sets the padding area on all four sides of an element
  • border - sets the border properties in one declaration
  • width, height - sets the width and height of an element
  • display - specifies how an element is displayed
